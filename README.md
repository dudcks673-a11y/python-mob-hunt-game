# Python Mob Hunt Game

## 프로젝트 소개
**Python Mob Hunt Game**은 파이썬의 클래스 상속(Class Inheritance) 및 객체 지향 프로그래밍(OOP) 개념을 실습하고 적용하기 위해 제작된 텍스트 기반의 간단한 몬스터 사냥 게임입니다. 플레이어는 용사가 되어 몬스터를 공격하고 물리치는 과정을 CLI(커맨드 라인 인터페이스) 환경에서 경험할 수 있습니다.

## 주요 기능
- **다양한 캐릭터 및 몬스터 구현**: `Player` 클래스와 공통 몬스터 부모 클래스 `Mob`을 상속하는 `Mushroom`, `Slime` 클래스를 통해 다양한 능력치와 고유 스킬(점프, 분열 등)을 가진 객체를 생성합니다.
- **전투 관리 시스템**: `BattleManager`를 활용하여 플레이어의 공격력과 몬스터의 방어력을 바탕으로 데미지를 계산하고, 체력 감소 및 사망 여부 판정을 수행합니다.
- **게임 루프 및 메뉴**: `Game` 클래스를 중심으로 반복 실행되는 메뉴 인터페이스를 제공하여 플레이어가 공격, 정보 조회, 게임 종료 등을 선택할 수 있도록 제어합니다.

## 프로젝트 구조
```
python-mob-hunt-game/
│
├── main.py          # 게임의 진입점(Entry Point). 플레이어/몬스터/배틀 매니저 생성 및 게임 시작
├── game.py          # 게임 메인 루프 실행 및 메뉴 입력 제어 (Game 클래스)
├── player.py        # 플레이어 캐릭터 정의 및 공격/정보 확인 기능 (Player 클래스)
├── monster.py       # 몬스터의 부모 클래스(Mob) 및 상속 클래스(Mushroom, Slime) 정의
├── battle.py        # 플레이어 공격 로직 및 몬스터 처치 판정 (BattleManager 클래스)
└── README.md        # 프로젝트 설명 문서
```

## 설치 방법
1. 이 저장소를 로컬 컴퓨터로 클론합니다.
   ```bash
   git clone https://github.com/<your-username>/python-mob-hunt-game.git
   cd python-mob-hunt-game
   ```
2. Python이 설치되어 있는지 확인합니다. (Python 3.8 이상 권장)
   ```bash
   python --version
   ```

## 실행 방법
1. 터미널 혹은 명령 프롬프트에서 아래 명령어를 실행하여 게임을 실행합니다.
   ```bash
   python main.py
   ```
2. 화면에 나타나는 메뉴 숫자를 입력하여 게임을 진행합니다.
   - `1`: 공격하기
   - `2`: 플레이어 정보 보기
   - `3`: 몬스터 정보 보기
   - `4`: 종료

## 팀원 역할
- **팀원 A**: `player.py`, `monster.py` 기초 설계 및 클래스 상속 구조(`Mob` -> `Mushroom`, `Slime`) 구현.
- **팀원 B**: `battle.py`의 `BattleManager`와 `main.py` 구현을 통한 객체 간 조율 및 전투 로직 검증.
- **팀원 C**: Git branch 협업 관리 및 최종 게임 연동 테스트 주도.

## AI Agent 활용
- **코드 자동 완성 및 리팩토링**: Antigravity AI Agent를 활용하여 `Game` 클래스의 메뉴 기반 루프 처리 로직을 안정적으로 설계하고 예외 상황(예: 입력 오류 및 Interrupt)을 처리했습니다.
- **연동 검증**: `player.py`, `monster.py`, `battle.py` 등 분할되어 있던 각 모듈 간의 의존성 및 메소드 호출을 올바르게 매핑하여 정상적인 게임 플레이 흐름을 구현하는 데 도움을 받았습니다.

## 향후 개선 사항
- **몬스터 반격 시스템 추가**: 플레이어의 일방적인 공격에서 벗어나, 몬스터도 턴에 맞춰 플레이어를 공격하고 플레이어의 체력이 소진되면 게임 오버되는 양방향 전투 시스템 구현
- **몬스터 다양화 및 특수 스킬 연동**: `Slime`의 분열(`split`), `Mushroom`의 점프(`jump`) 등 정의되어 있는 몬스터 특수 메소드를 실제 전투 턴에 적용
- **다양한 장비 및 아이템 상점**: 게임 내 획득한 골드(가칭) 등을 통해 공격력이나 방어력을 올릴 수 있는 성장 요소 구현
- **GUI 혹은 풍부한 CLI 효과**: 애니메이션 효과나 Rich 라이브러리를 활용해 콘솔 화면의 가독성과 연출 극대화
